6. Assessment Methodology

  • Completeness (20%) — Missing/null value analysis per column and row

  • Uniqueness (10%) — Duplicate row detection and key column integrity

  • Timeliness (10%) — Date ordering, temporal gaps, recency validation
  • Accuracy (15%) — Outlier detection via Z-score + Isolation Forest
